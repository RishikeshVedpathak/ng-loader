ng-loader Demo

Easy to integrate

<ng-loader theme="..."></ng-loader>

Simple to use

Use below tag in your html -
<ng-loader theme="default"></ng-loader>

Add/update theme list -
$scope.theme = ['default', 'theme1', 'theme2', 'theme3', ...];

Configure theme css -
.default{
background:linear-gradient(to right, rgba(47,160,209,1), rgba(128,128,128,1));
}
